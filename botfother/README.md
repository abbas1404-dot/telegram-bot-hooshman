# Telegram Bot with Welcome Message and Menu

بات تلگرام با پیام خوشامدگویی و دکمه‌های منو

## ویژگی‌ها (Features)

- ✅ پیام خوشامدگویی خودکار هنگام شروع بات
- ✅ دکمه‌های منو برای دسترسی سریع به محتوا
- ✅ رابط کاربری ساده و کاربرپسند
- ✅ پشتیبانی از زبان فارسی

## نصب و راه‌اندازی (Installation & Setup)

### 1. نصب وابستگی‌ها (Install Dependencies)

```bash
pip install -r requirements.txt
```

### 2. دریافت توکن بات (Get Bot Token)

1. در تلگرام به [@BotFather](https://t.me/BotFather) مراجعه کنید
2. دستور `/newbot` را ارسال کنید
3. نام و یوزرنیم بات را انتخاب کنید
4. توکن دریافتی را کپی کنید

### 3. تنظیم توکن (Configure Token)

فایل `.env` را ایجاد کنید و توکن خود را وارد کنید:

```bash
cp .env.example .env
```

سپس فایل `.env` را باز کرده و توکن را وارد کنید:

```
BOT_TOKEN=your_actual_bot_token_here
```

### 4. اجرای بات (Run Bot)

```bash
python bot.py
```

## استفاده (Usage)

1. بات را در تلگرام پیدا کنید
2. دستور `/start` را ارسال کنید
3. از دکمه‌های منو برای دسترسی به محتوا استفاده کنید

## سفارشی‌سازی (Customization)

می‌توانید محتوای منو را در فایل `bot.py` تغییر دهید:

- تغییر گزینه‌های منو در `MENU_OPTIONS`
- تغییر پیام‌های خوشامدگویی
- اضافه کردن گزینه‌های جدید
